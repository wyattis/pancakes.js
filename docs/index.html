▸ enablePhysics(opts, shape?)

Enable physics for this Sprite. Takes care of adding the body to the sprite.

enablePhysics(opts: object, shape: (undefined | string | Rectangle)?)
Parameters
opts (object)
Name Description
opts.mass float (default 100) The mass to use when bodies collide. Not sure how this number equates to real world mass.
opts.normalizedMaxSpeed float (default 0) The maximum, normalized speed of this body. If this value is 0 then the speed unlimited.
opts.maxSpeed Vector (default {}) Direction based max speed for the body.
opts.friction Vector (default {}) Friction to apply to the body.
opts.gravity Vector (default {}) Supplying a Vector will make gravity be applied for the given body.
opts.clamped boolean (default false) If true the Body will be clamped within the bounds of the world.
opts.fixed boolean (default false) If true the Body will not be affected by collisions it is a part of.
opts.enabled boolean (default true) If false then collisions won't happen with this body. This can be changed while the game is running by changing the sprite.body.enabled property.
shape ((undefined | string | Rectangle)?) This allows you to set your own geometry for this sprite. If left empty it will assume the shape is a rectangle the size of the image, tile or animation assigned to it.

Layer

Abstraction of a render layer. Each layer has its own canvas that is layered on top of the other canvases.

new Layer(scene: Engine.Scene, opts: Object)
Parameters
scene (Engine.Scene) Reference to the parent Engine.Scene
opts (Object) Options used when creating this Layer
Name Description
opts.depth integer (default 1) Specifies how much the layer should translate with the camera. Higher depth give the effect of the layer being further away. A depth of 0 indicates the layer should not translate at all. A depth of 1 makes the layer translate an equal amount.
opts.animated boolean (default true) Indicates if there are any animated or moving sprites in the layer. If this is set to false then the layer will only rerender when changes are made.
opts.zIndex Integer (default 0) Indicate where this layer should render in the z plane.
